Ejercicios

1.- En github tenéis la estructura de carpetas que necesitáis para hacer el ejercicio (una carpeta de ejercicios y una carpeta llamada Ejercicio_express).

2.- Crear un servidor con express dentro de la carpeta Ejercicio_express, cuyo archivo principal sea un index.js

3.- En el index.js he de tener 4 métodos get con rutas de entrada diferentes

4.- Uno de los métodos me tiene que enviar como respuesta un string que diga "hola, bienvenido a mi página web"

5.- Otro método me tiene que enviar como respuesta un html que contenga una barra de navegación con tres botones y que contenga además el código html de la web SocraDog

6.- Otro método me tiene que enviar como respuesta un html que contenga la misma barra de navegación y el código html de la web Forty (la de bootstrap)

7.- Otro método me tiene que enviar como respuesta un html que contenga la misma barra de navegación y el código html de la web Visualize.

8.- Los botones de la barra de navegación se llamarán SocraDog, Forty, Visualize

9..- Estos botones han de funcionarcorrectamente (si le hago click a SocraDog, se me mostrará socraDog, si le hago click a Forty, se mostrará Forty...)

10.- No quiero que las web tengan los estilos css marca_de_verificación_blanca

11.- Tampoco quiero que os traigáis a la carpeta de ejercicios, las carpetas de imágenes de las webs

Estos dos punto últimos aprenderemos a hacerlos el próximo día, por lo que las web se verán sin estilos y sin imágenes
